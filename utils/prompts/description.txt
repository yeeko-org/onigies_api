from django.contrib.auth.models import AbstractUser
from django.db import models


class User(AbstractUser):
    phone = models.CharField(max_length=100, blank=True)
    full_editor = models.BooleanField(
        default=False, verbose_name='Es capturista',
        help_text='Puede agregar notas, comentarios a los registros,'
                  'pero no tiene todos los permisos')
    mini_editor = models.BooleanField(
        default=False, verbose_name='Servicio social',
        help_text='Puede modificar ubicaciones y otros detalles')

    def get_full_name(self):
        if self.first_name and self.last_name:
            return f"{self.first_name} {self.last_name}"
        if self.first_name or self.last_name:
            return f"{self.first_name or self.last_name}"
        return self.username or self.email

    @property
    def is_full_editor(self):
        if self.is_anonymous:
            return False
        return self.is_superuser or self.is_staff or self.full_editor

    @property
    def is_admin(self):
        if self.is_anonymous:
            return False
        return self.is_superuser or self.is_staff

# **Esquema metodológico general** {#esquema-metodológico-general}

El primer aspecto de la metodología es la recabación de información de base sobre las IES, principalmente en torno a sus estructuras internas, sus poblaciones que integran su comunidad y sus formas de gobierno interno.

Posteriormente, la medición se realiza por **observable** y dos **variables**:

|  Observable | Variable A: contenido del observable armonizado e institucionalizado |
| :---- | :---- |
|  | **Variable B:** transversalización del observable |

El conjunto de observables conforman un listado, que **se estructura y se desagrega** de la siguiente manera:

| Eje 1 |  |  |  |
| ----- | ----- | ----- | ----- |
| Materia ↳ ↳  |  Componente ↳ ↳ |  ***Listado de observables:** da lugar a un cuestionario inicial de preguntas “Sí/No”* ⇩ | *Para las respuestas afirmativas en el cuestionario inicial: da lugar a un cuestionario específico, para conocer si el observable: \- Cuenta con ciertas características sustantivas (variable A) \- Cuál es el alcance del observable (variable B)* ⇩ |
|  |  |  |  |
|  |  | **Observable ↳ ↳** | Variable A: contenido del observable armonizado e institucionalizado |
|  |  |  | Variable B: transversalización del observable |
|  |  | **Observable ↳ ↳** | Variable A: contenido del observable armonizado e institucionalizado |
|  |  |  | Variable B: transversalización del observable |
|  | Componente ↳ |  **Observable ↳ ↳** |  Variable A: contenido del observable armonizado e institucionalizado |
|  |  |  | Variable B: transversalización del observable |
